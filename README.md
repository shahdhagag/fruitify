## Features
- Splash Screen, Home Screen, Cart Screen
- Display fruits and prices
- Simple and clean UI
- **Responsive UI** using `auto_size_text`
- **SVG support** using `flutter_svg`
- **Carousels/Sliders** using `carousel_slider`
- **Spacing & layout management** using `gap`
